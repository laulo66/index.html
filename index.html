ℹ️ ZezoLaunch

ZezoLaunch est un outil compagnon pour Virtual Regatta Offshore.


1. Copie tes coords depuis le journal de bord VR

2. Colle-les dans l'onglet Position

3. Appuie sur ZEZO ou consulte la Route


Format coords VR :
10°N 53'20" - 033°W 11'40"
